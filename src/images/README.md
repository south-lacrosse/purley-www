# Images

This folder contains original images in their best format, and also some intermediate forms.

* New lion logo
    * `PurleyNewLion-transparent.psd` - original
    * `PurleyNewLion-*` -  with various versions generated from that
    * `PurleyNewLion-with-text.psd` - with text as used in the original website, but not used in the current website as the logo is too wide
* Original logo
    * `Letterhead.docx` - the only copy we could find
    * `logo-original.bmp` - copied from the docx above, and should be the original form, so this should be a good image starting point
    * `logo-original.png` - saved with a transparent background.
* Unused

The favicons in `\public` were generated from `PurleyNewLion-transparent.png` using the [Real Favicon generator](https://realfavicongenerator.net/).

To resize and compress images you can use [Squoosh](https://squoosh.app/).
